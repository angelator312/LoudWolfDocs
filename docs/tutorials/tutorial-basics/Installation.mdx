# Installation

Open LoudWolf's [repository](https://github.com/angelator312/LoudWolf/tree/only_addons_dir)
And download ZIP

![img](/img/GithubDownloadZip.png)

Copy the `loud_wolf` folder to <code>your_project_path/addons</code>

Then open Godot,open Project/ProjectSettings.
And at plugins tab enable LoudWolf Plugin.

![img](/img/AddPluginInProjectSettings.png)

Then reload the Project to clear all errors.

:::tip
Keep reading the tutorial for better understanding of the plugin.
:::
